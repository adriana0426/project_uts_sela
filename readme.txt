SistemKaryawan_Sela
1) Pindahkan folder ini ke C:\laragon\www\ jika ingin menjalankan dengan Laragon.
2) Import file db_karyawan.sql via phpMyAdmin.
3) Cek includes/db_connect.php (default root tanpa password untuk Laragon).
4) Buka http://localhost/SistemKaryawan_Sela/login.php
